# H5 产品介绍长图方案

> 本文档描述一套 H5 长图风格的产品介绍图布局方案，适用于销售展示、产品推广等场景。
> 长图总长度约 9000-11000px，宽度 750px（标准手机屏幕宽度），分为 9 个内容区块。

---

## 整体设计规范

| 项目 | 规范 |
|------|------|
| 画布宽度 | 750px（移动端标准） |
| 配色方案 | 深蓝渐变背景（#1e1b4b → #4338ca）+ 白色卡片 + 翠绿/蓝色高亮 |
| 字体 | PingFang SC / Microsoft YaHei，标题 32-48px 加粗，正文 24-28px |
| 风格 | 简洁现代，大留白，卡片化布局，图标 + 数据驱动 |
| 图片风格 | 扁平化插画 + 产品截图 + 数据可视化图表 |

## 设计交付说明

| 项目 | 说明 |
|------|------|
| 最终交付格式 | 单张 PNG 长图（750px 宽） |
| 可编辑源文件 | Figma / Sketch / PSD |
| 字体包 | 随源文件附带 |
| 切图资源 | 图标（SVG）、截图（PNG 2x）单独导出 |
| 响应式适配 | 如需适配不同屏幕宽度，可导出 750px 和 1125px 两个版本 |
